## Cleanup File

Filename pattern:

```text
codex_session_cleanup_YYYY-MM-DD.md
```

Template:

```markdown
# Codex 会话删除说明（YYYY-MM-DD）

## 删除小结

用 3-5 个短段落说明本轮清理范围、删除判断、保留判断、无法处理会话的处理方式，以及是否存在需要用户手动确认的边界情况。

## 建议手动删除

| 会话 | 原因 | 删除信心 | 备注 |
|---|---|---|---|
|  | 已总结且无后续价值 | 高 |  |

## 建议保留

| 会话 | 保留原因 | 后续动作 |
|---|---|---|
|  | 仍有未完成任务 / 关键资产 / 用户明确保留 |  |

## 无法处理但建议清理

| 会话 | 问题 | 建议 |
|---|---|---|
|  | 缺失 / 读取失败 / 权限不足 / 索引残留 | 记录后删除 |

## 已执行后处理

- 已归档：
- 已加“已总结”标记：
- 未处理及原因：

## 不执行的动作

- 不永久删除当前控制线程。
- 不创建自动永久删除任务。
- 不删除未被记录到本说明里的会话。
```
